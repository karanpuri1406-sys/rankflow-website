# Requirements: AI-Powered SEO Agency (Chandigarh)

**Defined:** 2026-03-17
**Core Value:** SaaS and tech startups get faster organic growth through AI-powered SEO delivery that traditional agencies can't match on speed or cost.

---

## v1 Requirements

### Brand & Identity

- [ ] **BRAND-01**: Agency name defined with available .com domain (not already registered)
- [ ] **BRAND-02**: Positioning statement written targeting SaaS/tech startups ("The AI-powered SEO agency for SaaS")
- [ ] **BRAND-03**: Visual identity defined — color palette, typography, logo concept (minimalist style)
- [ ] **BRAND-04**: Agency tagline written (concise, memorable, AI-forward)

### Website

- [ ] **WEB-01**: Agency website built with Next.js + Vercel (free hosting)
- [ ] **WEB-02**: Homepage with hero, value proposition, services overview, social proof, CTA
- [ ] **WEB-03**: Services page with 3-tier SEO retainer packages and pricing
- [ ] **WEB-04**: About page with founder story, expertise, and AI-powered approach
- [ ] **WEB-05**: Blog/content section with CMS (Sanity free tier) for publishing SEO articles
- [ ] **WEB-06**: Contact page with lead capture form
- [ ] **WEB-07**: Case Studies / Results page (placeholder structure for first 3 case studies)
- [ ] **WEB-08**: Website is mobile-responsive and passes Core Web Vitals
- [ ] **WEB-09**: Website is SEO-optimized (meta tags, structured data, sitemap, robots.txt)
- [ ] **WEB-10**: Free SEO audit lead magnet integrated (form → email capture)

### Service Offerings

- [ ] **SVC-01**: Starter SEO package defined (scope, deliverables, pricing — ₹40-60K/month)
- [ ] **SVC-02**: Growth SEO package defined (scope, deliverables, pricing — ₹80-150K/month)
- [ ] **SVC-03**: AI-Powered Premium package defined (scope, deliverables, pricing — ₹150-300K/month)
- [ ] **SVC-04**: One-time Technical SEO Audit defined as a standalone entry-point product
- [ ] **SVC-05**: Service delivery checklists created for each package tier

### Client Operations

- [ ] **OPS-01**: Client intake form built (business info, goals, current SEO state, website access)
- [ ] **OPS-02**: Service agreement / contract template written (scope, timeline, payment terms, IP)
- [ ] **OPS-03**: Client onboarding email sequence written (welcome → kickoff → first week)
- [ ] **OPS-04**: Kickoff call agenda template created
- [ ] **OPS-05**: Monthly reporting template built (rankings, traffic, technical health, wins, next steps)

### Lead Generation

- [ ] **LEAD-01**: Agency website optimized to rank for target keywords ("SEO agency for SaaS", "SEO agency Chandigarh")
- [ ] **LEAD-02**: Free mini-audit lead magnet page built with email capture
- [ ] **LEAD-03**: LinkedIn agency page created and founder profile optimized for SEO agency positioning
- [ ] **LEAD-04**: Outreach message templates written for LinkedIn cold outreach to SaaS founders

### AI Toolstack

- [ ] **AI-01**: AI content workflow defined (Claude API integration for blog + client content drafts)
- [ ] **AI-02**: Content QA checklist created (AI draft → human review → fact check → publish)
- [ ] **AI-03**: SEO toolstack selected and set up (SE Ranking or Ahrefs, Google Search Console, Looker Studio)

## v2 Requirements

### Expanded Services

- **SVC-V2-01**: GEO (Generative Engine Optimization) add-on package — optimize for Google AI Overviews, ChatGPT Search, Perplexity
- **SVC-V2-02**: Programmatic SEO service for SaaS clients (landing page templates at scale)
- **SVC-V2-03**: Video SEO via partner videographer (YouTube optimization, video schema)
- **SVC-V2-04**: Performance/results-based pricing tier (retainer + traffic bonus)

### Operations & Scale

- **OPS-V2-01**: CRM set up (Zoho Free → Pipedrive at 10+ clients)
- **OPS-V2-02**: Automated reporting pipeline (Google Data Studio + Looker Studio dashboards)
- **OPS-V2-03**: Subcontractor/white-label network established for link building and content scale
- **OPS-V2-04**: International payment setup (Stripe / Payoneer for US/UK/AU clients)

### Team

- **TEAM-V2-01**: First hire defined and onboarded (SEO Specialist or Account Manager)
- **TEAM-V2-02**: Team SOPs documented for delegation of delivery tasks

## Out of Scope

| Feature | Reason |
|---------|--------|
| Paid ads management (PPC) | Scope creep — different skillset, different tools, fragments focus |
| Web design / development for clients | Not core SEO; refer out to partners |
| Social media management | Different service line; deferred to future |
| Mobile app | Web-first agency operations |
| Building proprietary SEO software | Use and integrate existing tools in v1 |
| Email marketing management | Out of scope for SEO agency positioning |
